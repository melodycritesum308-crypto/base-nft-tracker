# Base NFT Tracker

A beautiful, fast, and open-source NFT portfolio tracker built specifically for Base chain.

Track any wallet's NFTs on Base, view high-quality images, metadata, and collection details — all in a modern, responsive interface.

![Base NFT Tracker Preview](https://via.placeholder.com/1200x600/111827/ffffff?text=Base+NFT+Tracker+Preview)

## ✨ Features

- **Real-time NFT tracking** for any Base wallet address
- **High-quality image rendering** with IPFS support
- **Wallet connection** (view your own NFTs instantly)
- **Supports ERC-721 and ERC-1155** tokens
- **Clean, modern UI** built with Next.js 14 + Tailwind CSS + shadcn/ui
- **Fully responsive** design (mobile + desktop)
- **Fast data fetching** powered by Moralis Web3 API
- **Dark mode** optimized for crypto users

## 🛠 Tech Stack

- **Framework**: Next.js 14 (App Router) + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Web3**: wagmi + viem
- **NFT Data**: Moralis EVM API (excellent Base support)
- **Icons**: Lucide React
- **Deployment**: Vercel (recommended)
